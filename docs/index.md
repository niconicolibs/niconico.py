# niconico.py

niconico.py は、ニコニコ動画の情報取得や動画ダウンロードを扱う Python ライブラリです。

## 主な機能

- 動画情報の取得
- 動画の視聴ページ情報の取得
- コメント取得
- 動画ダウンロード
- ランキング、検索、マイリスト、シリーズの取得
- ログインユーザー向けの履歴、いいね履歴、投稿動画、フォロー情報の取得

## 対応範囲

2.2.x では、ニコニコ動画とユーザー関連の API を中心に対応しています。
静画、生放送、大百科は 2.3.0 以降で対応予定です。
